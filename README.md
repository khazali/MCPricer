# MCPricer
MCPricer is a Monte-Carlo options pricing tool, written in java.



## Usage
Just run MCPricer.jar located in as:

```bash
java -jar MCPricer.jar
```


## Assumptions
* A year is considered to be 365 days.
* Underlying stock pays no dividend.
